В работе над проектом я использовала Codex в основном по техническим моментам для реализации интерактивных механик и JavaScript-логики. С его помощью я собирала и дорабатывала анимацию первого экрана, механику лабиринта, перетаскивание флаконов в трафареты, игру с карточками, переключение дизайнов упаковки и рисование , а также финальный экран. Кроме этого, Codex помогал мне с адаптивной версией, CSS и JS, но визуальную композицию, расположение элементов и итоговые правки я принимала и подгоняла сама.
